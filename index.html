Produk Perdagangan Sistem Perdagangan Alternatif (SPA)

[cite_start]

Berdasarkan Undang-undang Nomor 10 Tahun 2011, SPA adalah sistem perdagangan yang berkaitan dengan jual beli Kontrak Derivatif selain Kontrak Berjangka, yang dilakukan di luar Bursa Berjangka secara bilateral dengan penarikan margin yang didaftarkan ke Lembaga Kliring Berjangka. [cite: 34, 35]

Contoh Transaksi: EU1010_BBJ (Daytrade) [cite: 107]

[cite_start]

Nasabah memprediksi kurs euro akan naik, lalu membeli 2 lot EU1010_BBJ di harga 1,1230. [cite: 108] [cite_start]Kemudian, posisi dilikuidasi pada harga 1,1240. [cite: 109]

Perhitungan Keuntungan:

P/L = [(Harga Jual - Harga Beli) x Contract Size x n Lot] - [(Facility Fee + PPN) x n Lot]
P/L = [(1,1240 - 1,1230) x 100.000 x 2] - [(30 + 3,3) x 2]
P/L = [0,0010 x 200.000] - 66,6
P/L = 200 - 66,6 = USD 133,4
 [cite_start]

Nasabah mendapat keuntungan sebesar Rp.1.334.000 (Fixed rate $1 = Rp 10.000) [cite: 114, 115]

[cite_start]

Contoh Transaksi: UJ1010_BBJ (Daytrade) [cite: 122]

[cite_start]

Nasabah memprediksi kurs USD/JPY akan turun, lalu menjual 1 lot UJ1010_BBJ di harga 142,20. [cite: 123] [cite_start]Kemudian, posisi dilikuidasi pada harga 142,05. [cite: 124]

Perhitungan Keuntungan:

P/L = [((Sell price - Buy price) / Liquidation Price) x Contract Size x n Lot] - [(Facility Fee + PPN) x n Lot]
P/L = [((142,20 - 142,05) / 142,05) x 100.000 x 1] - [(30 + 3,3) x 1]
P/L = 97,37 - 33,3
P/L = USD 64,07
 [cite_start]

Nasabah mendapat keuntungan sebesar Rp 640.700 (Fixed rate $1 = Rp 10.000) [cite: 129, 130, 131]

Perhitungan Keuntungan/Kerugian:

    [cite_start]
  • Keuntungan hari pertama: $976,7 [cite: 81]

  • Kerugian dari posisi pertama di hari kedua: -$318,3 [cite: 87]

  • Keuntungan dari posisi kedua di hari kedua: $1171,7 [cite: 90]
    • [cite_start]
  • Biaya Rollover Fee untuk 2 lot: -$10 [cite: 92]
Total P/L = $976,7 - $318,3 + $1171,7 - $10 = $1830,1 [cite_start]
