Contoh Transaksi: EU1010_BBJ (Daytrade)

Seorang nasabah memprediksikan kurs euro akan mengalami kenaikan, lalu ia mengambil posisi beli EU1010_BBJ pada harga 1,1230 sebanyak 2 lot. Tidak lama kemudian nasabah melikuidasikan posisi terbukanya pada harga 1,1240 sebanyak 2 lot.

Perhitungan Keuntungan:

P/L = [(Harga Jual - Harga Beli) x Contract Size x n Lot] - [(Facility Fee + PPN) x n Lot]
P/L = [(1,1240 - 1,1230) x 100.000 x 2] - [(30 + 3,3) x 2]
P/L = [0,0010 x 200.000] - [66,6]
P/L = 200 - 66,6 = USD 133,4

Nasabah mendapat keuntungan sebesar Rp.1.334.000 (Fixed rate $1 = Rp 10.000)
