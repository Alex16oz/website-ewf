[cite_start]P/L = [(36100 - 36200) x $5 x 1 lot] - [($30 + $3,3) x 1 lot] [cite: 62] [cite_start]P/L = (-100 x $5) - $33,3 = -$500 - $33,3 = -$533,3 [cite: 63, 64, 65]

Hari Kedua:

[cite_start]Posisi 1: P/L = [($3172,45 - $3175,30) x 100 ozt x 1 lot] - [$33,3] = -$285 - $33,3 = -$318,3 [cite: 85, 86, 87] [cite_start]Posisi 2: P/L = [($3172,45 - $3160,40) x 100 ozt x 1 lot] - [$33,3] = $1205 - $33,3 = $1171,7 [cite: 88, 89, 90]

[cite_start]$5 x 2 lot x 1 malam = $10 [cite: 92] [cite_start]
